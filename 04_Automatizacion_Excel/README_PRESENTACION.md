Procesamiento analítico de grandes volúmenes de datos en hojas de cálculo. Ideal para reportes, análisis de tendencias y gestión de bases de datos planas.

## 📦 Archivos
- `flujo_60_agente_excel.json`: Procesador de lógica de Excel.
- `flujo_60_subworkflow_excel.json`: Sub-procesos de datos.
- `datos_prueba_radon.xlsx`: Archivo de datos para validación.

## ✨ Características
- Lectura y escritura dinámica en Excel.
- Generación de reportes automáticos.
- Validación de integridad de datos.
